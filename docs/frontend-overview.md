# FilmGenX 前端设计概要

> 给 v0 的高层次设计文档
> 版本：v1.0 | 日期：2026-04-04

---

## 一、核心理念

**人机协作的分集动画生产系统**

- 每集动画 = 一个对话会话（Conversation）
- 所有对话内容永久保留，可随时回顾
- AI 生成剧本大纲 → 人工编辑 → 确认后进入分镜流程
- 每个阶段都可人工介入微调，不满意可重新生成

---

## 二、页面结构

```
登录页
  ↓
项目列表（卡片网格）
  ↓
点击某个项目 → 项目工作台
  ├── 分集列表（时间轴式）
  ├── 对话创作（进入聊天页创作新分集）
  ├── 角色管理
  └── 素材库
```

---

## 三、核心页面

### 1. 项目列表页 `/projects`

卡片网格布局，每个项目卡片显示：
- 封面图、项目名、原著名称
- 分集数量 / 完成度进度条
- [进入] 按钮

### 2. 项目工作台 `/projects/:id`

左侧导航 Tabs：
- **分集列表**（默认）
- 角色管理
- 素材库
- 项目设置

**分集列表内容**：
时间轴式展示所有分集，每个分集卡片显示：
- 分集编号、标题
- 状态徽章：对话中 / 已确认 / 制作中 / 已完成
- 进度条：分镜完成度 / 图像完成度 / 视频完成度
- 点击进入 → 分集详情页

顶部按钮：[+ 新建分集] → 打开对话页

### 3. 对话创作页 `/projects/:id/conversations/:id`

**这是创作一集的核心页面。**

布局：左右分栏（对话 70% | 信息 30%）

**左侧对话区**：
- 消息列表滚动区（用户消息 + AI 回复）
- AI 回复流式打字机效果
- 剧本大纲草稿以卡片形式内嵌在消息中（可展开/收起）
- 底部输入框 + [发送] 按钮

**右侧信息区**：
- 会话标题（可编辑）
- 当前状态：对话中 / 草稿待确认 / 已确认
- 快捷操作：
  - [总结剧本] — 触发 AI 生成大纲（流式输出）
  - 有草稿后显示 [再次总结] / [✓ 确认执行]
- 预设引导语下拉（一键发送常用提示）

**确认执行**：
点击 [✓ 确认执行] → 弹出对话框展示最终大纲 → 用户可编辑字段 → [确认] → 创建分集记录 + 触发分镜生成

### 4. 分集详情页 `/projects/:id/scenes/:sceneId`

**分集总控台。**

顶部面包屑：`项目 > 分集列表 > 分集详情`

**核心内容**：

```
┌─────────────────────────────────────────────────────────┐
│  第02集  药老现身 · 灵魂之约    状态：制作中             │
├─────────────────────────────────────────────────────────┤
│  本集概述：                                              │
│  萧炎在绝境中意外激活祖传戒指，神秘药老现身...           │
│                                                          │
│  生产流程：                                              │
│  ① 对话创作    ✓ 已完成    [查看对话]                   │
│  ② 分镜脚本    ⟳ 进行中    [进入分镜]                   │
│  ③ 图像生成    ○ 待开始    [批量生成]                   │
│  ④ 视频合成    ○ 待开始    [进入视频制作]               │
│                                                          │
│  快捷操作：                                              │
│  [编辑分集信息]  [继续对话]  [删除分集]                   │
└─────────────────────────────────────────────────────────┘
```

点击各个阶段的入口 → 跳转到对应页面

### 5. 分镜工作台 `/projects/:id/scenes/:sceneId/storyboard`

**左侧配置区（30%）**：
- LLM 配置下拉
- 提示词模板下拉
- 镜头数量滑块
- 风格备注文本框
- 系统提示词预览/编辑（可折叠）
- [触发 AI 生成] 按钮

**右侧分镜时间轴（70%）**：
- 情感曲线图（可折叠）
- 镜头卡片列表（可拖拽排序）
  - 每个卡片：镜头编号、时长、景别、动作描述、图像占位
  - 状态：草稿 / 已审核 / 已批准
  - 操作：[编辑] [生成图像] [删除]

**底部操作**：
- [+ 手动添加镜头]
- [批量生成图像]

### 6. 视频制作页 `/projects/:id/scenes/:sceneId/production`

**顶部镜头条**：横向滚动缩略图，点击切换当前镜头

**左侧当前镜头（50%）**：
- 镜头参数摘要
- 图像预览
- [修改提示词重新生成图像]
- 视频生成参数：质量 / 音效 / 起始帧
- [生成视频] 按钮

**右侧资源历史（50%）**：
- Tab 切换：[图像] [视频]
- 图像 Tab：历史版本网格，每张图可 [设为主图] [删除]
- 视频 Tab：视频预览 + 历史版本
- [生成新图像] / [生成新视频]

### 7. 角色管理页 `/projects/:id/characters`

角色卡片列表，点击进入角色详情：

**角色详情**：
- 左侧：基本信息（姓名、别名、关系）
- 右侧：版本时间线（v1 少年期 → v2 成年期 → v3 斗皇）
- 每个版本卡片：章节范围、外观描述、斗气颜色、参考图、base_prompt

### 8. 全局设置页 `/settings`

**LLM 配置**：
- 配置列表：名称 / Provider / Model / API Key
- [新建配置] → 填写表单 → [测试连接]

**提示词模板**：
- 按 workflow 分组：选段对话 / 剧本总结 / 分镜生成 / 视频提示
- 每个模板：名称、绑定模型、变量列表
- [新建模板] → 选择 workflow → 编辑系统提示词

## 四、关键交互流程

### 流程 1：创作新分集

1. 项目工作台 → [+ 新建分集]
2. 进入对话页，与 AI 讨论剧情
3. 点击 [总结剧本] → AI 生成大纲（流式输出）
4. 满意 → [✓ 确认执行] → 创建分集 → 自动跳转到分集详情
5. 不满意 → 继续对话 → 再次 [总结剧本]

### 流程 2：分镜微调

1. 分集详情 → [进入分镜]
2. 左侧配置参数 → [触发 AI 生成]
3. 右侧查看生成的镜头卡片
4. 点击某个镜头 [编辑] → 修改参数 → 保存
5. 点击镜头 [生成图像] → 查看生成结果
6. 不满意 → 修改 image_prompt → [重新生成图像]
7. 拖拽调整镜头顺序 → 自动保存

### 流程 3：视频制作

1. 分集详情 → [进入视频制作]
2. 顶部选择镜头
3. 右侧查看历史图像版本 → [设为主图]
4. 左侧调整参数 → [生成视频]
5. 等待任务完成 → 预览视频
6. 不满意 → 调整参数 → [重新生成视频]

## 五、技术要点

- **流式对话**：使用 `fetch` + `ReadableStream` 接收 SSE
- **消息类型渲染**：`text` 普通消息 / `outline_draft` 大纲卡片 / `outline_confirmed` 确认标记
- **拖拽排序**：镜头卡片支持拖拽，实时调用 API 保存顺序
- **任务轮询**：生成任务每 2 秒轮询一次状态，显示进度条
- **状态机**：分集状态流转（对话中 → 已确认 → 制作中 → 已完成）

## 六、UI 风格

- **主题**：暗色主题（深蓝 `#1a1f2e` + 金色 `#f0a500`）
- **字体**：中文用系统字体，代码/提示词用 monospace
- **动画**：页面切换 fade+slide，消息打字机效果
- **响应式**：优先 1440px 宽屏，最低 1024px
