# Types（TypeScript 类型定义）

与后端 Schema 对应的前端类型，保持字段命名一致（snake_case）。

| 文件 | 说明 |
|------|------|
| `project.ts` | Project 相关类型 |
| `scene.ts` | Scene、ScoreResult 类型 |
| `storyboard.ts` | Storyboard 类型 |
| `shot.ts` | Shot 完整类型（含 camera、character、environment 等嵌套结构）|
| `character.ts` | Character、CharacterVersion 类型 |
| `asset.ts` | Asset 类型 |
| `task.ts` | GenerationTask 状态类型 |
| `common.ts` | 通用类型（分页、API响应包装、枚举值）|
